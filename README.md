# Vegetables Project

## Description
Bu layihə onlayn tərəvəz və məhsulların idarə olunması üçün hazırlanıb. İstifadəçilər tərəvəzləri baxa, səbətə əlavə edə, sifariş verə və ödəniş edə bilirlər. Admin panel vasitəsilə məhsullar, kateqoriyalar və sifarişlər idarə olunur.
